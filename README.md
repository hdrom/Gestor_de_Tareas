La aplicación es un gestor de tareas sencillo que permite al usuario crear, editar y eliminar tareas personales, manteniéndolas organizadas en una lista interactiva.​
Cada tarea puede marcarse como completada o pendiente, y la información se guarda en una base de datos en la nube mediante Supabase, de modo que los datos quedan asociados al usuario autenticado y se sincronizan entre dispositivos
